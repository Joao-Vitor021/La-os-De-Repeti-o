# Laços de Repetição em Java

Projeto simples em Java criado para praticar o uso de laços de repetição.

## Descrição
O projeto demonstra o uso das estruturas `for` e `while` na linguagem Java.

- O laço `for` é utilizado para imprimir números de 0 a 10.
- O laço `while` é utilizado para imprimir números pares até 20.
